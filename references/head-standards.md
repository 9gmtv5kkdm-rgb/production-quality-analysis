# 封头制造标准族速查（production-quality-analysis 行业替换参考）

> 本文件供本技能在「压力容器/封头制造」语境下做缺陷归类与合规提示时调用。
> 仅列通用标准关系与映射，不替代正式设计计算与持证单位校审；具体条款以现行有效版本为准。

## 一、标准选型矩阵（工况 → 标准）

| 维度 | 主要适用标准 | 说明 |
|---|---|---|
| 压力容器通用 | GB/T 150.1~150.4—2011 | 材料 / 设计 / 制造 / 检验通用要求 |
| 钢制封头 | JB/T 4746—2019 | 封头型式、尺寸、公差、成形要求 |
| 焊接工艺评定 | NB/T 47014（原 JB/T 4708） | 焊接工艺评定 |
| 承压设备焊接 | NB/T 47015 | 焊接规程 |
| 产品焊接试板 | NB/T 47016 | 试板与性能检验 |
| 不锈钢 / 复合板 | NB/T 47010 | 不锈钢和耐热钢 |
| 焊接材料 | NB/T 47018 | 焊接材料订货技术条件 |
| 无损检测 | NB/T 47013.1~13 | RT/UT/MT/PT/ET… 方法与验收级别 |
| 安全技术监察 | TSG 21—2016 | 固定式压力容器；NDT 比例、耐压试验 |
| 制造许可 | TSG 07—2019 | 特种设备生产单位许可规则 |

## 二、缺陷族 → 标准映射（用于归因报告的"合规提示"列）

| 缺陷族 | 关联标准 / 条款要点 | 合规提示 |
|---|---|---|
| 减薄量（F 坡口/修磨/减薄） | JB/T 4746 成形减薄、最小厚度；GB/T 150 名义厚度 | 标注"减薄是否超成形极限"，建议核 JB/T 4746 |
| 成形方法（D 变形/鼓包） | JB/T 4746 冷/热成形边界 | 提示冷成形 vs 热成形的判定 |
| NDT 比例（E/G 焊缝） | TSG 21 第 4 章、NB/T 47013 验收级 | 标注"焊缝检测比例与验收级别"，不可代判合格 |
| 材料（H 裂纹/材质） | 材料质保书、NB/T 47018 焊材 | 提示批次追溯与质保书核对 |
| 尺寸（A/B 直径/圆度） | JB/T 4746 外形公差 | 标注外形公差依据 |
